# silva_gold

The `silva_gold()` function creates a silva based reference for use with
the [`rchime()`](http://mothur.org/rchime/reference/rchime.md) function
in reference mode. It contains 5,181 sequences.

## Usage

``` r
silva_gold()
```

## Value

data.frame containing selected rchime parameters and their values.
